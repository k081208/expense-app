# 経費管理アプリ

エクセルへの手打ちをやめて、スマホから経費を記録できるようにする個人用アプリです。

- 課金なし・外部の経費管理サービスなし
- データは自分の Google アカウント内（Google スプレッドシート + Google Drive）にのみ保存
- iPhone の Safari から「ホーム画面に追加」してアプリのように使える(PWA)
- オフラインでも入力でき、電波が戻ったら自動で同期

## できること

- 日付・カテゴリ・金額・メモ・レシート写真を記録
- カテゴリは自由に追加・削除
- 月ごとの一覧・合計金額
- カテゴリ別内訳グラフ、直近6か月の推移グラフ
- CSV エクスポート（Excelで開けます）

## 使い始めるまでの手順

大きく2つのステップです。**手順1は最初の1回だけ**行えば大丈夫です。

1. Google Cloud Console で「OAuth クライアントID」を作る
2. アプリを開いて初期設定 → iPhone のホーム画面に追加する

### 手順1: Google Cloud Console で OAuth クライアントIDを作る

このアプリがあなたの Google スプレッドシートと Google Drive にアクセスするための「鍵」を発行します。無料で作成できます。

1. [Google Cloud Console](https://console.cloud.google.com/) にアクセスし、Googleアカウントでログイン
2. 画面上部のプロジェクト選択から「新しいプロジェクト」を作成（名前は「経費管理アプリ」など何でもOK）
3. 左メニュー「APIとサービス」→「ライブラリ」を開き、以下の2つを検索して**両方とも「有効にする」**をクリック
   - `Google Sheets API`
   - `Google Drive API`
4. 左メニュー「APIとサービス」→「OAuth同意画面」を開く
   - User Type は「外部」を選択して作成
   - アプリ名（例: 経費管理アプリ）、ユーザーサポートメール、デベロッパーの連絡先メールを入力して保存
   - 「スコープ」の画面で「スコープを追加または削除」から以下を検索して追加
     - `.../auth/spreadsheets`
     - `.../auth/drive.file`
   - 「テストユーザー」の画面で、**自分の Google アカウントのメールアドレス**を追加
   - 公開ステータスは「テスト」のままで問題ありません（自分専用アプリのため審査は不要です）
5. 左メニュー「APIとサービス」→「認証情報」を開く
   - 「認証情報を作成」→「OAuth クライアント ID」を選択
   - アプリケーションの種類は「ウェブ アプリケーション」を選択
   - 「承認済みの JavaScript 生成元」に次のURLを追加: `https://k081208.github.io`
   - 「作成」をクリックすると **クライアントID**（`xxxxxxxx.apps.googleusercontent.com` の形式）が表示されるのでコピーしておく

### 手順2: アプリを開いて初期設定

1. スマホ（またはPC）のブラウザで https://k081208.github.io/expense-app/ を開く
2. 「初期設定」画面が表示されるので、手順1でコピーした**クライアントID**を貼り付けて「保存して次へ」
3. 「Google でログイン」をタップし、経費管理アプリ用に作成した Google アカウントでログイン・許可する
   - 「Google は このアプリを確認していません」という警告が出ることがありますが、自分で作った個人用アプリなので想定どおりです。「詳細」→「(アプリ名)に移動」で進んでください
4. 初回ログイン時に、自動で以下がGoogleドライブに作成されます
   - スプレッドシート「経費管理アプリデータ」（経費データとカテゴリ一覧）
   - フォルダ「経費管理アプリ_レシート」（レシート写真の保存先）

### iPhone のホーム画面に追加する

1. Safari でアプリのURLを開く
2. 共有ボタン（□に↑のアイコン）をタップ
3. 「ホーム画面に追加」を選択

ホーム画面のアイコンから開くと、アドレスバーのないアプリのような画面で使えます。

## データはどこに保存される？

すべて **あなたの Google アカウントの中だけ** に保存されます。

- 経費データ・カテゴリ一覧 → Google スプレッドシート「経費管理アプリデータ」
- レシート写真 → Google Drive フォルダ「経費管理アプリ_レシート」

このアプリ自体はサーバーを持たず、ブラウザから直接あなたの Google アカウントにアクセスするだけなので、開発者や第三者がデータを見ることはありません。スプレッドシートは設定画面の「Googleスプレッドシートを開く」からいつでもPCで直接確認・編集できます。

## オフラインでの動作について

電波が悪い場所で記録した経費は端末内に一時保存され、「未同期」と表示されます。オンラインに戻ると自動的に同期されます。設定画面の「今すぐ同期」でも手動同期できます。

## トラブルシューティング

- **ログインできない**: 設定画面の「クライアントIDを変更する」からクライアントIDを確認してください。OAuth同意画面の「テストユーザー」に自分のアカウントを追加し忘れていないかも確認してください。
- **同期されない**: 設定画面で「未同期」の件数を確認し、「今すぐ同期」をタップしてください。それでも失敗する場合は一度ログアウトしてログインし直してください。
- **別の端末でも使いたい**: 同じGoogleアカウントで同じURLを開けば、同じスプレッドシート・データを引き続き使えます。
